# Lead Tracker - Google Extension

A simple Chrome extension to save and manage links while browsing.

## Features

- Save the URL of the current tab with one click
- Manually enter and save any URL
- Persists data in `localStorage` so leads survive browser restarts
- Double-click to delete all saved leads

## Usage

| Action | How |
|---|---|
| Save current tab | Click **SAVE TAB** |
| Save a custom URL | Type in the input field and click **SAVE INPUT** |
| Delete all leads | Double-click **DELETE ALL** |
